Primera entrega

Proyecto CRUD escuela secundaria

La idea con este proyecto era presentar un ejemplo de CRUD (create, read, update, delete) basandome en una escuela secundaria ficticia de Uruguay. Por el momento decidí centrarme en elementos que ya hemos utilizado en clase, como por ejemplo, variables, funciones, métodos de funciones, entre otras cosas.

Lo que he intentado con esta entrega es crear una base de datos funcional que permita almacenar datos de estudiantes así como también agregar y realizar otras acciones que emulen el comportamiento de una app en la vida real.
